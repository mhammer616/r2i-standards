Online Resources

Below you will find some good online resources that we use to learn from and stay up to date with latest trends and ideas.

W3Schools

Probably one of the best and most informative websites for help with remembering how to format an HTML, CSS, JavaScript, or jQuery tag is W3schools.com.

W3C

Want an even deeper dive into the spec? Go to the source at W3.org.

Also on w3.org's website we can do HTML validation and CSS validation.

HTML5 Please

HTML5 Please is an online resource that allows you to search for the best way to use various tags that are in development by the browser, and what the support of that feature may be. It supplies information on the status of HTML, CSS, and JavaScript features. You can use this to determine whether you need to worry about adding something to modernizr.js to detect if you need to alter styles or JavaScript to make something work.

Can I Use?

Can I Use? is an online resource that allows you to search for the availability of HTML, CSS, and JavaScript Features as well as see their acceptance.

Codrops

A great inspiration and example site that supplies various ways of implementing behaviors. Take some time to explore, learn, and use some of the effects found on codrops.

CSS Tricks

CSS Tricks is a blog that has a lot of good resources and information on developing CSS. Offering snippets, an almanac, videos, and forums to help develop better CSS.

Smashing Magazine

Smashing Magazine is a great online, and offline, read to keep up to date with trends and ideas going on in Design and Development

A List Apart

A List Apart is another good online resource to keep up to date with trends and ideas going on in Design and Development

CSS Zen Garden

Originally created in 2001 as a way to show of what CSS was capable of and used as a resource to sell business owners of the benefit of switching to CSS, CSS Zen Garden, evolved into a way to show off creativity as well. Over 218 designs have been created, all of them use the exact same HTML and you are only allowed to edit the CSS and image files used. Today they are re-working the HTML to be more semantic and removing wrapping elements that were put in place to give people something to style.
