Basic Hero Guidelines

These go by a lot of different names, so we want to try and unify these different names and use the term Hero. The different terms you may hear are home campaign, landing banner, page banner, banner, header, page header, or hero.

The hero generally contains a background graphic, color, image or video, a heading, and subheading. Depending on SEO's recommendation, the heading text may or may not be the <h1> tag, always check with them before placing the <h1> in the hero.

The hero may also contain a carousel that rotates through various slides. Using semantic markup, you should place a <header> tag as a direct descendent of the <main> tag to place the hero inside, then style based on that hierarchy. All the examples below will use that as a base.

Images and Videos in Heroes

Sitecore, Adobe, Drupal, WordPress and Static Sites should use this approach for images:

Images should be done as background images, which probably means using inline styles wight background-image to apply it. Then in the CSS files we can do everything else with background properties that is necessary, such background-size: cover; and background-position: center center; or background-position: center top;.

DNN should use this approach for images:

If the previous approach is not possible within the CMS, then use an <img /> tag and position the image absolute and size appropriately using CSS. If taking this approach, make sure the image properly scales within the wrapping element and doesn't appear less than the height or width. This can be done using position: absolute; left: 50%; top: 50%; transform: translate(-50%, -50%); min-height: 100%; min-width: 100%;.

Placing videos as a background:

Match the aspect ratio of the video to the wrapping element and apply using padding-top. Absolute position all child elements, including the video. The video should have the following properties set: height: 100%; width: 100%; left: 0; top: 0; position: absolute;.

The video should also be set to loop and have no audio track so that it doesn't annoy people when your page loads.
