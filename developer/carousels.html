Basic Carousel Guidelines

Carousels are used on practically ever website we create. While we have used a number of different carousel libraries, including jCarousel, Slick and Bootstraps very own Carousel plugin, we have found that the most flexible library is Owl Carousel. Owl Carousel prvodies many options & events to acheive practically any carousel we use. In addition, the responsive set up is simple to use and already has touch swipe enabled by default.
