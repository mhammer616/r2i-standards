JSCompress

JSCompress is an online javascript compressor that allows you to compress and minify your javascript files. Compressed javascript files are ideal for production environments since they typically reduce the size of the file by 30-80%.

Lorem Ipsum

No, that's not just placeholder text I forgot to delete, there are several Lorem Ipsum generators out there, some are fun and will put lyrics from Snoop Dogg, or generate using names of cat breeds, but while those can be fun, most of the time they aren't very professional looking. Please stick with Lorem Ipsum Feed to generate the dummy content for our websites.

Need some dummy images, use lorempixel.com to generate some images in color or gray.

Codepen

An online playground to show off your latest creation and get feedback, build examples to show a bug that you can't get out there, find examples of design patterns, or just send someone a code snippet, CodePen is the perfect tool to use.

Favicon Generator

Create all favicons using this generator which will output all four sizes needed for the favicon. X-icon Editor lets you preview, and edit, what the favicon will look like at 64x64, 32x32, 24x24, and 16x16. You can even create an icon from scratch.

Quick Diff

Quick Diff is an online tool that allows you to compare two files online and see what and where the differences are.
