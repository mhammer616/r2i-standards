Basic Tabs Guidelines

Almost all the sites we develop use tabs somewhere on them, to keep tabs consistent in functionality and code base, we have chosen to use Bootstraps Tabs as our standard and best practice. The Bootstrap Tab component is easy to setup and works with all of our CMS's. Go to getbootstrap.com/javascript/#tabs for additional documentation on how this can be customized. A few examples of layouts and styles have been added below along with the code to achieve their look and feel.
